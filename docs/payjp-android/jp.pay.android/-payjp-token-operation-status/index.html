enum PayjpTokenOperationStatus : Enum<PayjpTokenOperationStatus>

The client status of token request.

Provide a guide as to whether or not UI should allow user's submits to prevent too many post request in a short period by client side.

You can request to create a token regardless of this status, but it is not recommended.
